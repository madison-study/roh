# RSA 암호화란 

RSA는 공개키 암호 방식 중 가장 널리 쓰이는 알고리즘

두 키(public, private)을 사용하여 데이터를 암호화/복호화합니다

* 공개키(public key): 암호화에 사용
* 개인키(private key): 소유자만 알고 있음

## 소인수분해

RSA는 큰 소수의 곱셈의 소인수 분해가 어렵다는 점을 그 기반으로 삼는다. (모듈러 역원의 계산이 불가능에 가까움)

## 알고리즘

1. 서로 다른 두 소수 p,q 의 선택

eg: p = 61, q = 53

2. n = pq 계산

RSA의 modulus

n = 61 * 53

3. 오일러 피 함수 φ(n)=(p−1)(q−1) 계산

φ(n)=(61−1)(53−1)=60×52=3120

4. 공개 지수 e 선택

조건
* 1 < e < φ(n)
* gcd(e,φ(n)) = 1

예: e = 17

5. 개인 지수 d 계산

d는 다음 관계를 만족함:

        e × d ≡ 1 mod φ(n)

즉 d는 e의 모듈러 역원

예: 𝑑 = 2753

6. 최종 키 구성

* pub: (n, e) = (3233, 17)
* private: (n, d) = (3233, 2753)

7. 사용

* 암호화

평문 m 
    c ≡ m^e mod n

* 복호화
    m ≡ c^d mod n
